# CraftyPrint-E-Commerce-Template-Marketplace
Crafty Print is a Filipino-based online shop specializing in educational and learning materials. Built with modern web technologies, Firebase Firestore, and EmailJS, it offers a seamless shopping experience with features like product browsing, search, AI-powered suggestions, cart management, reviews, admin dashboard, and more.
Features

Responsive Product Grid: Showcases products with images, ratings, views, sales, and prices.  
Filter & Search: Category filtering, keyword search, and AI-powered suggestions.  
Wishlist: Save products for later.  
Cart & Checkout: Manage items, view totals, and submit orders via GCash.  
Product Details: Detailed view with images, descriptions, reviews, and related products.  
Reviews System: Submit and display reviews, integrated with Discord for moderation.  
Admin Panel: Manage products, orders, analytics, and top products.  
Localization: Multi-language support including English, Tagalog, Bisaya, Ilocano, Japanese, Korean, Chinese, and Arabic.  
Currency Selector: Switch between PHP, USD, EUR, GBP, JPY, KRW, SGD, AUD, SAR.  
Music Player: Background music with playlist and volume control.  
Dark Mode: Toggle between light and dark themes.  
Notifications: Toast messages and announcement banners.  
User Engagement: Abandoned cart popup, admin access via double-click, and dynamic content loading.

Structure & Components

Header: Logo, site title, language selector, theme toggle, cart icon with count.  
Admin Panel: Password-protected; manage orders, view analytics, add/delete products.  
Categories & Search: Filter by category, keyword search, sorting options.  
Product Grid: Dynamic product cards with images, ratings, actions.  
Modals: Product details, reviews, support, order tracking, wishlist, add product, currency, instant checkout, achievements.  
Music Player: Background music controls and playlist.  
Footer: Developer info and version.

Usage

Setup: Ensure Firebase configuration is correct; add your products in Firestore.  
Admin Access: Double-click logo; enter password (CraftyPrintX1@).  
Manage Products: Add or delete via admin modal.  
Order Management: Confirm payments, update statuses, upload download links.  
Customer Interaction: Search, filter, view products, add to cart/wishlist, leave reviews.  
Theme & Music: Toggle dark mode and control background music.

Technologies

HTML, CSS, JavaScript  
Firebase Firestore for data storage  
EmailJS for notifications  
Discord Webhook integration for moderation

Customization

Update Firebase config with your project details.  
Replace Discord webhook URLs for orders, reviews, and support.  
Add/edit products via Firestore or admin modal.  
Modify language translations and currency options as needed.
